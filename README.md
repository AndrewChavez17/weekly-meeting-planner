# Weekly Meeting Planner

A Python program that helps users organize weekly meetings by assigning leaders and discussion topics. The program allows users to view all meetings or filter for unassigned (open) meetings.

## Features
- Set up a customizable number of weekly meetings
- Assign a leader and topic to each meeting
- View all meetings or only unassigned meetings
- Menu-driven interface for ease of use
- Modular design using reusable functions

## Technologies Used
- Python
- Functions with parameters
- Lists
- Loops
- Conditional logic
- User input handling

## How to Run
1. Ensure Python is installed
2. Download or clone this repository
3. Run the program:
   ```bash
   python meeting_planner.py
